Sydney Sister
=========

Project 2 WDI6

This is an e-commerce web app for women's fashion.


Gems
--------------

```
gem 'bcrypt'
gem 'hiredis', '~> 0.4.5'
gem 'rails_12factor'
gem 'mail'
gem 'stripe'
gem 'foundation-rails'
```

API
---------------

- Stripe API for payment gateway
- Mail API for sign up confirmation
